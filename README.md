# soroban-i-like-big-budgets

To install dependencies:

```bash
bun install
```

To prepare:
```bash
./docker-testnet.sh
```

To deploy:

```bash
bun run deploy.ts
```

To run:

```bash
bun run index.ts
```

---

1. Start with running `index_1.ts` under `./docker-testnet.sh`
2. Next run `index_2.ts` under `./docker-testnet.sh`
3. Finally run `index_3.ts` under `./docker-unlimited.sh`
